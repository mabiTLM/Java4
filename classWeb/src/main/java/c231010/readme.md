## 서버관리자

  - 보통은 백엔드들의 당번제로 처리한다.
  - 왜 당번제?
  
## 총괄
 - PM, 포르젝트 매니저
 - 프로젝트를 진행함에 있어서 전반적인 모든 사항을 확인하고 관리한다.
 - 개발자가 언제까지 작업하는지, 언제 확인할 수 있는지 등
 - 디자인이 언제 나오는지, 개발자에게 전달됐는지 등
 - 전체적으로 프로젝트가 얼마나 걸리는지 등
 
## Back End

 - 기능구현 개발자

 - Java, Javascript, golang, C++, C, Ruby, ...
   - Java -> Spring, Javascript -> Node.js, golang -> 개념
 - DB까지 관리한다.
